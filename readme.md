# Boletin Noticias MVC

## Programado en Java con el Framework Spring
La aplicación Web tiene un menú principal con los siguientes items:

* Lista de Periodistas 
* Lista de Agencias
* Lista de Noticias

### Apartado de Periodistas
Se despliega una lista de los periodistas existentes, en la columna Detalle se puede ver los datos en detalle de cada periodistas y realizar la actualización de los datos que se deseen.
En la columna Acción con el botón Borrar se puede eliminar el periodista que se desee.
Con ayuda del botón Agregar, se pasa a un formulario en el cual se introducen los datos de un nuevo periodista. 

* El siguiente link <http://localhost:8080/periodista.htm> es para acceder a las funcionalidades de periodistas
